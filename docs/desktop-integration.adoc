= Setup desktop integration

Freedesktop conforming graphical desktop interfaces like Gnome on Ubuntu or KDE on Kubuntu support the following to create a desktop integration.

Relative paths are relative to the root dir of the project.

== Install desktop integration

Execute the following with low privileges e.g. from inside a distroless and rootless container:

[source,bash]
----
node desktop-integration/create-integration.mjs desktop-integration/compile_html_viewer
----

Execute the following with elevated privileges:

[source,bash]
----
cp desktop-integration/HTMLSlideshow.desktop /usr/share/applications/HTMLSlideshow.desktop
cp desktop-integration/compile_html_viewer /usr/bin/compile_html_viewer
chmod a+x /usr/bin/compile_html_viewer
cp desktop-integration/mime-package.xml /usr/share/mime/packages/x-embed-html.xml
update-desktop-database /usr/share/applications/
update-mime-database /usr/share/mime/
----

== Remove desktop integration

[source,bash]
----
rm /usr/share/mime/packages/x-embed-html.xml
rm /usr/share/applications/HTMLSlideshow.desktop
rm /usr/bin/compile_html_viewer
update-desktop-database /usr/share/applications/
update-mime-database /usr/share/mime/
----
